# Movie Data Analytics & Handwritten Digit Recognition Platform

A full-stack data visualization platform built with **Spring Boot**, **Vue 3**, **MySQL**, **ECharts**, and a **Python neural network** for handwritten digit recognition.

---

## Features

### Movie Data Analytics

- User registration and login (JWT Authentication)
- TMDB Top 100 movie crawler
- Interactive data visualization dashboard
- Top rated movies
- Movie release year distribution
- Movie language distribution
- Movie genre distribution
- Movie table with detailed information

### AI Handwritten Digit Recognition

- Pure Python implementation of a feedforward neural network
- MNIST dataset training
- Image upload from web interface
- Spring Boot calls Python model
- Real-time prediction results
- Confidence score display

# Database Setup

Create a new MySQL database:

```sql
CREATE DATABASE movie_platform;
```

Modify

```
backend/src/main/resources/application.properties
```

Example:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/movie_platform
spring.datasource.username=root
spring.datasource.password=YOUR_PASSWORD
spring.jpa.hibernate.ddl-auto=update
```

Start Spring Boot.

JPA will automatically create the required tables.

# Handwritten Digit Recognition

## Download MNIST Dataset

Download the CSV version of MNIST:

https://pjreddie.com/projects/mnist-in-csv/

or

https://www.kaggle.com/datasets/oddrationale/mnist-in-csv

Place the files into

```
python_model/mnist_dataset/
```

The directory should look like

```
python_model
└── mnist_dataset
    ├── mnist_train.csv
    └── mnist_test.csv
```

---

## Install Python Dependencies

```bash
cd python_model

pip install -r requirements.txt
```

---

## Train the Model

```bash
python train.py
```

After training,

```
model.pkl
```

will be generated automatically.

---

## Run the Web Application

### Backend

```bash
cd backend

mvn spring-boot:run
```

### Frontend

```bash
cd frontend

npm install
npm run dev
```

Open

```
http://localhost:5173
```

# Notes
